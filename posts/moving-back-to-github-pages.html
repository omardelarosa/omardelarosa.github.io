omardelarosaJune 13th 2018, 10:30:11 pm

A while back, I made a big deal about moving this site to Cloudflare and using HTTPS. Well now that Github Pages supports HTTPS for custom domains it's time I moved back (Ultimately, Ilove the convenience of GH Pages.)

This will obviously have no functional difference on this page. But in case you were wondering, it's back on GH Pages and the code/content for this site lives entirely in a static repo.
